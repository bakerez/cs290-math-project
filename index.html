Cool Math Visualizations

The purpose of this website is to explain some concepts in advanced mathematics or computer science. We would like to use some fancy visualizations or other creative means to show off how these concepts work. We will rework this main page as the project progresses. Some ideas of topics we could do are:

  • Singular value decomposition and image compression (Ezra)
  • Fourier analysis (continuous and/or discrete Fourier transforms)
  • Differential equations
  • Linear algebra/matrix methods
  • Topics in calculus (single or multivariate)
